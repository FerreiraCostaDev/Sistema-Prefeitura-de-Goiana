# Sistema-de-Gest-o-de-Planos-de-A-o---Prefeitura-de-Goiana
Este projeto está sendo desenvolvido pelo Squad 34 (FICR - Recife) com o objetivo de digitalizar a gestão de políticas públicas da prefeitura. O sistema visa substituir os processos em papel por uma plataforma web que permite o monitoramento de prazos, controle de secretarias e vinculação com as 17 ODSs.
